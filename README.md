# aws_restaurant_chatbot

Cloud Computing assignment designed to take user input and return restaurant suggestions.

AWS microservices utilized include:
- S3
- API Gateway
- Lambda
- DynamoDB
- Lex
- OpenSearch (ElasticSearch)
- SQS
- SES/SQS
- CloudWatch
